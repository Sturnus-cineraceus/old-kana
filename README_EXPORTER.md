# CSV Exporter仕様

このプロジェクトで使用する CSV エクスポーターについての仕様を記述します。

- 実行スクリプト: `python scripts/export_csv.py`
- 出力ファイル: `out/mapper.csv`
- 出力カラム: `平仮名,UNICODE`

各フィールドの取り扱い
- 平仮名
  - テーブルの各行にある 2 番目の `<td>`（index 1）をタグ除去してそのまま出力する。
  - 例: `<td>あ</td>` → `あ`
- UNICODE
  - その行の HTML（tr 要素内）に存在する最初の `<code>` タグの inner text を出力する（例: `U+1B003`）。
  - `<code>` タグが無ければ空欄を出力する（空文字）。
  - HTML 内のフォントや CSS 表記（`@font-face` 等）はコードポイントへ変換しない。
- その他
  - 3 列目（変体仮名グリフなど）はエクスポート対象外。
  - HTML の記述順序を尊重し、正規化や重複除去は行わない。

実行例
```
python scripts/export_csv.py
```

注意事項
- スクリプトはリモートのテーブル（https://cid.ninjal.ac.jp/kana/list/）を取得してパースします。ネットワーク環境に依存します。
- 出力ファイルは `out/mapper.csv` に上書きで保存されます。既存ファイルのバックアップが必要な場合は事前に保存してください。
